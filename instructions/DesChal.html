Design Challenge Test



This test is minutes long.

Your team has been asked to help with the design for a meeting scheduler. The scheduler should help suggest a suitable time and place for meetings taking into account the following:

1) Meeting significance
2) Participant importance
3) Participant flexibility
     - Which dates participants are free
     - Which dates participants are busy
4) Which location people are likely to be in
5) Which rooms and resources are available to book

Task 1 - Draw up storyboards (with stickmen) and write descriptions of possible problems that such a system could face.

Task 2 - Draw up possible solutions to these problems and describe using storyboards

Please use a seperate set of storyboards for each problem or solution and write the description on the provided paper.
